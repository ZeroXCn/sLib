# sLib库
<span>
</p>
<br>
一个C++库,初衷是用来作为Simple2D的基础类库,后来添加许多功能,并且简单封装了WinAPI的一些函数
</p>

<br>Examp:存放实例的地方
<br>sCore:核心库,包含字符处理,时钟,线程,日期等
<br>sData:数据库,一般处理各种数据库,跨各种数据库连接
<br>sFile:文件库,处理文件,文件夹
<br>sGraphic:图形库,基于GDI
<br>sHelper:辅助库,给予其他各个部分辅助用
<br>sIO:输入输出库(流库),处理IO的部分
<br>sMath:数学库,各种数学上的算法等
<br>sMedia:媒体库,播放声音等媒体
<br>sMisc:混合库,乱七八糟的东西
<br>sNet:网络库,处理网络信息
<br>sOS:操作系统库,一些关于操作系统部分
<br>sParser:解析库,主流信息格式解析,包括Json,Xml等
<br>sPattern:模式库,设计模式
<br>sUi:界面库,基于sGraphic开发,Win32 API封装

</span>

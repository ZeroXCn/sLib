# sLib库
Examp:存放实例的地方
sCore:核心库,包含字符处理,时钟,线程,日期等
sData:数据库,一般处理各种数据库,跨各种数据库连接
sFile:文件库,处理文件,文件夹
sGraphic:图形库,基于GDI
sHelper:辅助库,给予其他各个部分辅助用
sIO:输入输出库(流库),处理IO的部分
sMath:数学库,各种数学上的算法等
sMedia:媒体库,播放声音等媒体
sMisc:混合库,乱七八糟的东西
sNet:网络库,处理网络信息
sOS:操作系统库,一些关于操作系统部分
sParser:解析库,主流信息格式解析,包括Json,Xml等
sPattern:模式库,设计模式
sUi:界面库,基于sGraphic开发,Win32 API封装
